# MERN Notely 📝

Notely is a full-stack note-taking web application built using the MERN stack. It enables users to securely create, view, update, and delete personal notes after authentication.

## 🚀 Features
- Create, edit, and delete notes
- Notes accessible only to the authenticated user
- RESTful API architecture
- Clean and responsive UI

## 🛠 Tech Stack
- **Frontend:** React, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Atlas)
- **Version Control:** Git & GitHub
